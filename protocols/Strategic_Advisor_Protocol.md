# Strategic Advisor Protocol: AI as Thinking Partner

**Purpose:** Transform AI from a task executor into a strategic thinking partner. Use this framework to ensure you're asking the right questions, not just getting quick answers.

**Created:** February 16, 2026  
**How to use:** Reference this protocol at the START of important sessions. Answer the pre-session questions BEFORE engaging AI.

## ðŸŽ­ Tone & Personality Control

**Why it matters:** Explicitly setting tone gets better results. Different situations need different AI "voices."

### Tone Options

Pick one per session (or per message):

- **Direct / No-fluff** â€” Fast, high signal, minimal explanation
- **Socratic / Challenger** â€” Pushes assumptions, asks hard questions
- **Executive / Polished** â€” Professional, presentation-ready
- **Mentor / Supportive** â€” Teaching + encouragement + reasoning
- **Analyst / Precise** â€” Exact, detail-oriented, evidence-based
- **Creative / Exploratory** â€” Open-ended, brainstorming, possibilities
- **Technical / Implementation** â€” Step-by-step, debugging-focused

### Tone Modifiers

Add these when needed:
- **"Be concise"** â€” Short bullets, key points only
- **"Be thorough"** â€” Full reasoning and evidence
- **"List risks first"** â€” Prioritize red flags
- **"Give options, then recommend"** â€” Show choices before advice
- **"Ask me 2-3 clarifying questions"** â€” Ensure alignment before proceeding

## ðŸ§  Advanced Prompting Framework

### The 5-Line Structure

Use this for complex requests:

1. **Goal:** What you need
2. **Mode:** AI collaboration mode
3. **Tone:** Voice you want
4. **Constraints:** Length, format, requirements
5. **Output:** What to deliver

**Example (Strategic Critique):**
```
Goal: Critique my approach to [project]
Mode: Strategic Thinking
Tone: Direct, no-fluff
Constraints: 5 bullets max, list risks first
Output: Weaknesses + blind spots + reframe + next steps
```

**Example (Executive Document):**
```
Goal: Create 1-page proposal for [initiative]
Mode: Building (with strategic layer first)
Tone: Executive, polished
Constraints: One page, clear headings, minimal jargon
Output: Proposal + 3-sentence executive summary
```

## ðŸ”„ Iterative Collaboration Patterns

### Pattern 1: Critique â†’ Refine â†’ Validate
1. **First prompt:** "Here's my approach. What's wrong with it?"
2. **AI critiques:** Identifies weaknesses
3. **Second prompt:** "I've addressed X and Y. What about Z?"
4. **AI validates:** Confirms improvements or identifies remaining issues

### Pattern 2: Options â†’ Analysis â†’ Decision
1. **First prompt:** "Here are 3 options. Compare them."
2. **AI analyzes:** Provides comparison matrix
3. **Second prompt:** "Given [constraints], which do you recommend?"
4. **AI recommends:** With reasoning and implementation path

### Pattern 3: Draft â†’ Challenge â†’ Finalize
1. **First prompt:** "Create draft of [document]"
2. **AI drafts:** Initial version
3. **Second prompt:** "Now critique this draft as if you're [stakeholder]"
4. **AI challenges:** Identifies weaknesses
5. **Third prompt:** "Revise based on critiques"

## âš ï¸ Red Flags: When to Pause and Rethink

**Stop and reconsider if:**

- [ ] You're asking AI to build without first asking it to critique your approach
- [ ] You're accepting AI's first answer without challenging it
- [ ] You're treating AI like a search engine instead of a thinking partner
- [ ] You haven't told AI what you're WORRIED about
- [ ] You're asking for a deliverable without explaining WHY you need it
- [ ] You're in "task mode" instead of "strategic mode"
- [ ] You're avoiding the hard questions you should be asking
- [ ] You're building something without validating the need first

**Better approach:**
1. Share your concern or uncertainty first
2. Ask AI to help you think through it
3. THEN ask for the deliverable

## ðŸ“Š Session Management Best Practices

### Start of Session
1. State your goal clearly
2. Share your biggest uncertainty
3. Tell AI what mode you need
4. Set expectations for output

### During Session
- **Challenge AI's answers:** "What if [alternative]?"
- **Ask follow-up questions:** "Why?" "What else?" "What am I missing?"
- **Be specific:** "Give me 3 examples" not "Give me examples"
- **Redirect when off-track:** "That's not quite what I meant. Let me clarify..."

### End of Session
1. Summarize key decisions
2. Document action items
3. Note what to follow up on later
4. Save important exchanges for reference

If AI gives you a bad/generic/unhelpful response:

**Don't just re-prompt. Diagnose first:**

- [ ] Was my goal unclear?
- [ ] Did I provide enough context?
- [ ] Did I ask the wrong question?

- [ ] Am I actually asking the right thing?

**Then improve:**
```

Output: [exact format/structure you want]"
```
